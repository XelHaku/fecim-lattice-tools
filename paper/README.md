# FeCIM Lattice Tools Paper

This directory contains the in-repository LaTeX draft for the FeCIM Lattice Tools tools paper.

## Status

Current status: early scaffold. The draft is intentionally conservative: unsupported numeric claims are described as validation targets or omitted until a reproducible artifact exists.

## Build

```bash
bash paper/build.sh
```

The build script prefers `latexmk`. If `latexmk` is unavailable, it falls back to `pdflatex` plus `bibtex`.

Output:

```text
paper/main.pdf
```

## Source Files

| Path | Purpose |
|---|---|
| `main.tex` | Main LaTeX manuscript |
| `refs.bib` | Seed bibliography |
| `PLAN.md` | Six-week execution plan and claim gates |
| `figures/` | Generated paper figures and source scripts |
| `data/` | Paper-specific data snapshots and provenance notes |
| `tables/` | LaTeX tables included by `main.tex` |
| `reviews/` | Self, AI, and peer review notes |
| `drafts/` | Saved draft milestones |

## Public-Safety Rules

- Do not commit private addresses, private reviewer names, or personal contact details.
- Do not claim device measurement unless measured data and provenance are committed.
- Do not claim final accuracy, energy, area, or timing numbers unless generated by a reproducible script.
- Keep limitations visible in the abstract, validation, and conclusion.

## Citation Source Of Truth

The long-term source of truth for bibliography work is [citations/](../citations/README.md). The current `paper/refs.bib` file is a seed bibliography for the early LaTeX draft; migrate durable source records into `citations/papers/` before relying on them for final paper claims.
