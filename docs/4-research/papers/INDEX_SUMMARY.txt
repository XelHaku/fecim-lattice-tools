CONTRIBUTING NEW PAPERS
================================================================================

1. Find PDF from arXiv, journal, or preprint
2. Download and save to _tools/downloaded/{source}/
   (source = arxiv, nature, springer, ieee, other)
3. Identify correct topic using TOPIC_SUMMARIES.md
4. Move to: by-topic/{NN}-{topic}/ with descriptive name
5. Update: PAPERS_INDEX.md with entry
6. Commit: git add ... && git commit -m "docs: Add paper on {topic}"

File naming: {descriptor}_{year}.pdf or {Author}_{year}_{descriptor}.pdf
Examples:
  - first_principles_hfo2_superlattice_2024.pdf
  - Tung_2025_Modeling_and_Design_Enablement.pdf
