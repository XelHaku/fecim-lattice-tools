# Validation Documentation

This directory groups validation references by responsibility:

- `claims/` — scientific claim matrices and benchmark evidence.
- `contracts/` — cross-module physics contracts and source-of-truth mappings.
- `policies/` — validation confidence, trust boundaries, and verification policy.
- `baselines/` — baseline and periodic-review records.
- `module1/` and `module4/` — module-specific observation and investigation records.
- `audits/` — cross-module depth audits.
- `external/` — external benchmark references.
- `reviewer/` — reviewer-facing quick-start and research-grade testing standards.
- `tools/` — feasibility notes for external validation tooling.
