## Or Create Your Own Reference File

```markdown
# docs/sky130-reference/SKY130_QUICK_REFERENCE.md

# SKY130 Quick Reference for FeCIM Design

Source: https://skywater-pdk.readthedocs.io/

## Standard Cell Dimensions (sky130_fd_sc_hd)

| Parameter | Value | Notes |
|-----------|-------|-------|
| Cell Height | 2.72 μm | 8 metal tracks |
| Site Width | 0.46 μm | unithd site |
| Site Name | unithd | For LEF SITE |

## Metal Layer Stack

| Layer | Name | Min Width | Min Space |
|-------|------|-----------|-----------|
| 1 | li1 | 0.17 μm | 0.17 μm |
| 2 | met1 | 0.14 μm | 0.14 μm |
| 3 | met2 | 0.14 μm | 0.14 μm |
| 4 | met3 | 0.30 μm | 0.30 μm |
| 5 | met4 | 0.30 μm | 0.30 μm |
| 6 | met5 | 1.60 μm | 1.60 μm |

## Power Rails

| Rail | Layer | Width |
|------|-------|-------|
| VPWR | met1 | 0.48 μm |
| VGND | met1 | 0.48 μm |

## Pin Placement Guidelines

- Use met1 for cell pins (WL, BL)
- Power pins on top/bottom edges
- Signal pins centered vertically

## References

[1] Google/SkyWater. "SKY130 Open Source PDK Documentation."
    https://skywater-pdk.readthedocs.io/en/main/
    Accessed: 2026-01-25

[2] SKY130 Standard Cell Library (sky130_fd_sc_hd)
    https://github.com/google/skywater-pdk-libs-sky130_fd_sc_hd
```
