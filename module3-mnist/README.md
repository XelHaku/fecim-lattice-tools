# Module 3: MNIST

Module 3 provides MNIST inference/training paths for FeCIM hardware-aware simulation.

## Scope
- Floating-point and CIM inference modes
- Quantization/noise sweeps across discrete levels
- Dataset loading and GUI comparison workflows

## Key paths
- `pkg/core/` – inference, quantization, metrics
- `pkg/gui/` – visualization, controls, dual-mode UI
- `pkg/training/` – training and weight export utilities
