# Module 1: Hysteresis

Module 1 implements ferroelectric hysteresis modeling for FeCIM materials.

## Scope
- Preisach and Landau-Khalatnikov (L-K) engines
- ISPP/WRD target progression and calibration workflows
- GUI + headless execution paths

## Key paths
- `pkg/core/` – physics/model logic
- `pkg/gui/` – module UI and state orchestration
- `cmd/` and root launcher integration – CLI entry points
