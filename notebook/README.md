# Lab Notebook

This directory is for public engineering and validation notes.

Rules:

- Keep entries factual.
- Do not include private outreach names, private addresses, private emails, or confidential feedback.
- Record what was tried, what worked, what failed, and what should happen next.
- Link to issues, PRs, validation artifacts, and commits where possible.

Suggested filename format:

```text
YYYY-MM-DD.md
```
